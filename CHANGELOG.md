# Changelog

All notable changes to the JGFall Dev Toolkit will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-01

### Added

#### Components
- React Button component with multiple variants
- React Card component with flexible layouts
- React Modal component with accessibility features
- React component export index

#### Styling Framework
- Spacing utilities (margin, padding)
- Flexbox utilities
- Design tokens for colors
- CSS variables for theming
- Dark mode support

#### Deployment Scripts
- Netlify deployment script
- Vercel deployment script
- GitHub Pages deployment script
- All scripts with colored output and error handling

#### Setup Tools
- React app scaffolding script
- Static site generator
- Git hooks setup script
- Pre-commit hooks (linting, formatting)
- Pre-push hooks (testing)

#### Build Tools
- Image optimization script
- Support for JPG and PNG optimization

#### Utilities
- JavaScript storage helpers (localStorage/sessionStorage)
- JavaScript validators (email, URL, phone, password, credit card)
- JavaScript date utilities (formatting, relative time)
- Python file utilities (read/write JSON, YAML)
- Python CLI helpers (colors, prompts, progress bars)

#### Templates
- README.md template
- CONTRIBUTING.md template
- .env.example template
- package.json template

#### Documentation
- Comprehensive README.md
- Contributing guidelines
- Changelog
- Code examples
